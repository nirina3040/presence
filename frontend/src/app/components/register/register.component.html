Comment ça marche ?

  1. Remplissez les informations de l'étudiant
  2. Cliquez sur "Démarrer l'enregistrement"
  3. Regardez la caméra (30 photos seront prises)
  4. Le modèle est automatiquement entraîné
  5. L'étudiant peut maintenant être reconnu !

Notes importantes

  • Assurez-vous que la caméra est bien connectée
  • Bonne luminosité pour une meilleure reconnaissance
  • Visage bien visible et sans obstacles
  • L'ID étudiant doit être unique
